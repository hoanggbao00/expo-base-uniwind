## Translations (i18n)

### What the repo uses

- `i18next` + `react-i18next`
- Typed resource keys and namespaces under `src/i18n/locales/`

### Use the typed hook

Prefer **`useAppTranslation()`** from `src/i18n/use-app-translation.ts`.

It supports two call shapes:

- **Single namespace**: `useAppTranslation("common")` then `t("someKey")`
- **Multiple namespaces**: `useAppTranslation(["common", "settings"] as const)` then `t("common:someKey")`

Notes:

- The hook returns `{ _, t }`
  - `_` is the raw `react-i18next` `t` function
  - `t` is the typed wrapper that returns `string`

### Resource typing

- Resource key types come from `src/i18n/locales/resources.ts`.
- When adding new namespaces or keys, update resources and JSON locale files under `src/i18n/locales/`.
